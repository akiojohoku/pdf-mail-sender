# PDF個別メール送信システム

名簿CSVと全員分がまとまったPDFを読み込み、通し番号ごとにPDFを分割して、
各生徒(保護者)のメールアドレスへ個別にGmail経由で送信するWebアプリです。

> **💡 ブラウザだけで使える「GAS版」もあります** → [gas/](gas/) フォルダ
> 職員室PCでの起動・Pythonのインストール・アプリパスワードが不要になります。
> 新しく導入する場合はGAS版がおすすめです。以下はPython版(職員室PC方式)の説明です。

## 動作の仕組み

- 職員室のPC 1台でこのアプリを起動しておくと、他の先生は校内ネットワーク経由で
  ブラウザからアクセスして使えます(起動時に表示されるURLを開くだけ)。
- 送信はGmailのサーバー(SMTP)へ直接接続して行います。各先生は送信のたびに
  「自分のGmailアドレス+アプリパスワード」を入力します(サーバーには保存されません)。

## 初回セットアップ(アプリを起動するPCのみ・1回だけ)

1. **Python 3 をインストール**(入っていなければ)
   - https://www.python.org/downloads/ からダウンロードしてインストール
   - **Windowsの場合は、インストール画面で「Add python.exe to PATH」に必ずチェック**
2. このフォルダ(`pdf_mail_sender`)を起動するPCにコピー
3. 起動ファイルをダブルクリック
   - Mac: `起動_Mac.command`
     (初回は Finder で右クリック →「開く」が必要な場合があります)
   - Windows: `起動_Windows.bat`
   - 初回はライブラリの自動インストールに1〜2分かかります

## 毎回の起動

起動ファイルをダブルクリックするだけです。黒い画面に

```
このPCから使う場合   : http://127.0.0.1:8787
他のPCから使う場合   : http://192.168.x.x:8787
```

と表示され、ブラウザが自動で開きます。他の先生には「他のPCから使う場合」の
URLを伝えてください。**黒い画面を閉じるとアプリが終了します。**

※他のPCからつながらない場合は、起動しているPCのファイアウォールで
Python(ポート8787)の通信を許可してください。

## Gmailアプリパスワードの発行方法(先生ごとに1回)

1. https://myaccount.google.com → 「セキュリティ」
2. 「2段階認証プロセス」を有効にする(すでに有効ならそのまま)
3. 検索窓で「アプリ パスワード」を探して開く
4. アプリ名(例: PDF送信)を入力して「作成」→ 表示された **16文字** をメモ
5. このアプリの STEP 4 でその16文字を入力(スペース入りのままでもOK)

※アプリパスワードは Google アカウント本体のパスワードとは別物です。
※発行画面が見つからない場合は、学校のGoogle Workspace管理者に
「アプリパスワードの利用許可」を依頼してください。

## 使い方

1. **STEP 1**: 自分のメールアドレスを入力(保存済みテンプレートが読み込めます)
2. **STEP 2**: 名簿CSV・PDF・1人あたりのページ数を指定して「読み込んでチェック」
   - CSVの列: `通し番号,クラス,出席番号,生徒氏名,メールアドレス`
     (画面の「CSVテンプレートをダウンロード」からひな形を取得できます)
   - 通し番号1の生徒 → PDFの先頭からNページ、通し番号2 → 次のNページ…と割り当て
   - PDFの総ページ数が「人数×N」と一致しないと送信できません
   - メールアドレスが空欄の行は自動でスキップされます
3. **STEP 3**: 宛名(君/保護者様)・件名・添付ファイル名の頭・本文を入力
   - 「テンプレートとして保存」で次回から呼び出せます(アドレスごとに保存)
4. **STEP 4**: アプリパスワードを入力し、
   まず「自分宛てにテスト送信」で内容を確認 → 問題なければ「本送信を開始」
   - 約1秒間隔で1通ずつ送信されます(300名で5〜10分程度)
   - 途中で「中断」できます。失敗した人だけの再送信もできます
5. 画面右上の「送信履歴」で過去の送信結果を確認できます

## 注意事項

- Gmailの送信上限(Google Workspaceは1日約2,000通)にご注意ください
- 送信履歴には宛先・件名・成否のみ記録され、本文やPDFの中身は保存されません
- `data/` フォルダにテンプレートと履歴が保存されます
