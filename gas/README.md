# PDF個別メール送信システム(GAS版)

Google Apps Script(GAS)で動く「ブラウザだけ」版です。
職員室PCでの起動・Pythonのインストール・Gmailアプリパスワードは**すべて不要**。
各先生はURLを開き、Googleにログインしているだけで自分のアカウントから送信できます。

## Python版(親フォルダ)との違い

| | Python版 | GAS版(このフォルダ) |
|---|---|---|
| サーバー | 職員室のPCで起動 | 不要(Googleのクラウド) |
| アプリパスワード | 必要 | **不要** |
| 送信元 | 入力したアドレス | ログイン中の自分のアカウント |
| テンプレート | サーバーに保存 | 自分のGoogleアカウントに保存 |
| 送信履歴 | サーバーに保存 | 自分のGoogleスプレッドシートに自動保存 |
| 送信上限 | Gmail準拠 | 1日あたり約1,500通(Workspace) |

## 設置手順(管理する先生が1回だけ・15分程度)

1. https://script.google.com を開く(学校のGoogleアカウントで)
2. 「**新しいプロジェクト**」をクリック
3. 左上の「無題のプロジェクト」をクリックし、名前を「**PDF個別メール送信システム**」に変更
4. エディタに最初から開いている `コード.gs` の中身を全部消し、
   このフォルダの **`Code.gs`** の中身を貼り付ける
5. 左の「ファイル」の「+」→「**HTML**」→ 名前を **`index`** にして作成し、
   中身を全部消して **`index.html`** の中身を貼り付ける
6. 💾(保存)を押す
7. 右上の「**デプロイ**」→「**新しいデプロイ**」
   - 歯車アイコン → 種類:「**ウェブアプリ**」
   - 次のユーザーとして実行: 「**ウェブアプリにアクセスしているユーザー**」 ← 重要!
   - アクセスできるユーザー: 「**(学校のドメイン)内の全員**」
   - 「デプロイ」をクリック
8. 表示された **ウェブアプリのURL**(`https://script.google.com/a/macros/～/exec`)をコピーして、
   使う先生方に共有する(ブックマーク推奨)

## 先生が初めて使うとき(先生ごとに1回)

1. 共有されたURLを開く
2. 「承認が必要です」と表示されたら「権限を確認」→ 自分のアカウントを選択 → 「**許可**」
   - これは「あなたのアカウントからメールを送る」「履歴シートを作る」ことへの同意です
3. 以降は開くだけで使えます

## 使い方

Python版と同じ流れです(STEP 1〜4)。違いは:

- STEP 1 に自分のアカウントが自動表示されます(パスワード入力なし)
- 送信したメールは自分のGmailの「送信済み」に残ります
- 送信履歴は自分のGoogleドライブに「PDF個別メール送信システム_送信履歴」
  というスプレッドシートで自動保存されます

## 注意事項

- **送信が終わるまでブラウザの画面を閉じないでください**(数名ずつ順番に送信されるため)
- 送信上限は1人の先生あたり1日約1,500通です(画面のSTEP 1に残り数が表示されます)
- コードを修正したときは「デプロイ」→「**デプロイを管理**」→ ✏️ →
  バージョン「**新バージョン**」→「デプロイ」で反映されます(URLは変わりません)
